# RSS Feed Reader

This web application is used to parse RSS feed URLs using Nokogiri and show the latest articles sorted by date.

## Getting Started

* Ruby version  - 2.6.1

* Rails version  - 5.2.3

* System dependencies 

* Configuration

* Database creation

  ```
  rails db:create
  rails db:migrate
  ```

* Database initialization

  ```
  rails db:seed
  ```

* How to run the test suite

  ```
  rails test:controllers
  rails test:models
  rails test:system
  ```
